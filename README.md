本系统想完成：个性化学习平台。初始根据学习体检来形成学习画像和专属学习计划（接入千问），后根据用户使用这个平台的风格，不断补充完善形成新的学习画像和计划，数据库也自动记录用户的使用对话，好的对话可加入学习快照日后查看，同时拥有历史记录回溯。样式方面参考deepseek等ai平台，风格为新春，因为本人做这个系统的时候正值新春（可换风格）。
注意：本系统未完善
